Vanilla JS - Login based Album Gallery App (Using Ajax / Json)


1. Make login screen with form validation
2. On submit login form, using AJAX check username and password combination from users.json file.
3. On successful login store user info in session storage or local storage.
4. After login redirect to albums page and show albums grid. Fetch album data using ajax from json file and display as grid.
5. On click any album, show images grid of the album using ajax and json.
6. On clicking any image show image in popup.
7. User should not be able to access albums and images page without login. 
8. User once logged in should not be asked to login again on refreshing page.
9. Add logout link on album and images pages. On click logout user and redirect to login form page.
